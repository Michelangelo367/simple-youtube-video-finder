# simple-youtube-video-finder
get public video data from youtube.

|supported ?| param name|
|-----------|-----------|
| x  | part(required)   |
| x  | channelId |
| x  | channelType  |
| x  | maxResults |
| x  | order |
| x  | pageToken |
| x  | published_after |
| x  | published_befor |
| x  | q |
| x  | type |
|    | regionCode |
| x  | relevanceLanguage |
|    | safeSearch  |
|    | location |
|    | locationRadius |
